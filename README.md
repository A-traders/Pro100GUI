# Pro100GUI

Автономный GUI-оркестратор для пайплайна Pro100 поверх MT5 Strategy Tester.

Заменяет ручную последовательность скриптов (`opt_pro100`, `mm_sweep`, `build_pro100_pdf`)
на единое приложение с конфигурацией, прогрессом, возобновляемостью и просмотром результатов.

## Запуск

```
pythonw Pro100GUI.pyw
```

При первом запуске приложение само установит недостающие зависимости
(см. `bootstrap.py`). Требуется только установленный Python 3.11+.

## Архитектура

5 слоев, снизу вверх:

1. **core** -- чистая логика, без GUI и MT5. Модели конфигурации,
   расчет периодов, фильтры, TWR-математика, кодек pro100 csv.
2. **adapters** -- внешние системы. `TerminalRunner` (запуск
   `terminal64.exe`), `SetFileBuilder`, `EARegistry`, `FilesStaging`,
   `PdfRenderer`, `PdfQC`, `EAFetcher` (скачивание актуального
   советника из публичного Telegram-канала).
3. **orchestrator** -- DAG задач, состояние, JSON-persistence,
   EventBus.
4. **gui** -- PySide6, 4 экрана: Конфигурация / Прогон / Результаты /
   Настройки.
5. **app** -- shell, логирование, точка входа.

## Источник советника

Файл советника не хранится в репозитории и не используется локально с диска.
При каждом запуске приложение проверяет последнюю версию в публичном
канале <https://t.me/xauruspro/16> и подкачивает обновление.
